帮助：com.sea.ftp.boot.BootStartup [OPTION] [PARAMS]
	FTP服务器帮助文档。

	[OPTION]包含命令start|stop|suspended|resume|help
		start		启动FTP服务器。
		stop  		关闭FTP服务器。
		suspended	暂停FTP服务器。
		resume		恢复FTP服务器。
		help		查看帮助文档。
		
	[PARAMS]命令参数（可选）
		-default		使用默认配置。
		-config	[file]	指定服务器配置。
		-h [host]		指定服务器绑定的IP。
		-p [port]		指定服务器绑定的端口。
		-ssl			是否使用SSL。
	例子：
	com.sea.ftp.boot.BootStartup start 使用默认配置启动FTP服务器。
	com.sea.ftp.boot.BootStartup start -config ./conf/config.xml 使用config.xml中的配置启动FTP服务器。
	com.sea.ftp.boot.BootStartup start -h 127.0.0.1  启动FTP服务器绑定IP为127.0.0.1 。
	com.sea.ftp.boot.BootStartup start -p 21 启动FTP服务器,端口为21。
	com.sea.ftp.boot.BootStartup start -ssl 启动FTP服务器,使用SSL。
